- ```java.lang.UnsatisfiedLinkError: Can't load library: /usr/lib/jvm/java-11-openjdk-amd64/lib/libawt_xawt.so``` 
On a Ubuntu machine try to install the Java 11 JDK:

```
sudo apt install openjdk-11-jdk
```
